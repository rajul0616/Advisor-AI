One-time setup
AdvisorAI uses Google OAuth. You need a free Google Cloud Client ID to enable sign-in.
  1. Go to console.cloud.google.com/apis/credentials
  2. Create a project, then click + Create Credentials → OAuth client ID
  3. Choose Web application. Under Authorised JavaScript origins add:
    https://rajul0616.github.io
  4. Click Create, copy the Client ID (ends in .apps.googleusercontent.com)
  5. Paste it below and click Save & Continue
⚠Please enter a valid Client ID.

Session ends when you close the browser tab.
Signed-in user data is never sent to any server.
